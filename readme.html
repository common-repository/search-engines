A smart plugin that helps you to optimize the blog for SEO purposes.

First

If you operate your website for hobby or profit, SEO can be an important tool in making your website popular. SEO is not rocket science (or anywhere close to it). But it certainly can get as technical and detailed as you want to make it.

This plugin is here to help you get your job done!

— Luis Alberto Ochoa Esparza

Installation

  1. Unzip the search-engines.zip folder.
  2. Upload the search-engines folder to your /wp-content/plugins directory.
  3. In your WordPress dashboard, head over to the Plugins section.
  4. Activate Search Engines.

Features

  1. Title control
  2. Meta descriptions
  3. Meta keywords
  4. Avoid duplicate content
  5. Canonical URLs
  6. Protect RSS Feeds
  7. Custom Post Types
  8. Taxonomies
  9. And More...
